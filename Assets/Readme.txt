Кратакое описание к проекту:

1) Структура менеджеров выстроена атким образом, чтобы подменеджеры были частью одного большого менеджера
это дает возможность менять компоненты не меняя интерфейс взаимодействия

2) Скрипт XMLAdapter является адапрером между XML данными и менеджерами

3) Подменеджеры реализуют интерфейс ISubManager, который доступен через namespace "AmconNS"

4) Для повторного использования объектов используется пул, который в случае нехватки компонентов
спавнит недостающие
